播放器单次播放设置原型

目标是给播放器添加一个“播放设置”按钮，点击后打开一个小型设置面板，用于调整本次播放的 速度 和 播放模式。下面给出 3 个原型方向。

方案 A · 紧凑下拉菜单

推荐

播放速度

播放模式

Direct
启动更直接，适合浏览器可直接播放的源。
HLS
兼容性更强，适合需要 remux / transcode 的场景。
优点：最符合“小菜单”的需求，视觉最轻，不会把底栏挤得太复杂。适合当前播放器的沉浸式方向。

HLS (Remux)
当前会话更偏向兼容性，支持连续 seek 与 session 切换。
更适合想展示更多说明文案的版本。

优点是信息更清楚，缺点是比菜单感更重，更像一个小卡片。
