侧边栏「继续播放」入口原型

这份原型只验证全局入口的位置、层级和文案。它不是 mini player，也不表示离开播放页后视频仍在后台播放。 推荐语义是「继续播放」，让用户快速回到刚才的播放页。

展开侧边栏：后端状态上方的可点击卡片

主页面仍然保持浏览任务，提示只占用底部全局状态区。

设计意图

用户离开播放器后，入口固定出现在全局状态区；它比后端状态更靠上，说明优先级更高，但仍然不打断浏览。

收起侧边栏

icon-only + tooltip

侧栏收起时不显示标题

推荐规则

  • 只在离开播放页后显示，同一播放器页面内隐藏。
  • 点击后带上最新 `t` 和 `autoplay=1`。
  • 播放结束、接近结尾或手动关闭后隐藏。
  • 收起态只保留 icon，完整信息放到 tooltip / aria-label。

不推荐文案

  • 首版不默认写「正在播放」，因为离开播放页后视频实际不会继续后台播放。
  • 首版不展示完整观看历史，避免变成第二个历史入口。

移动端抽屉：复用展开卡片

移动端打开导航抽屉时，继续播放入口仍在后端状态上方。
