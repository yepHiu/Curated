# Curated v1.3.1 Release Notes Draft

## Summary

Curated v1.3.1 focuses on library workflow polish and playback immersion. This release adds actor external-link management, makes curated-frame export format configurable with JPG support, and tightens the in-player immersive viewing experience with edge-to-edge playback and quieter hidden-chrome feedback.

## Highlights

- Added actor external-link storage and UI so actor profile details can surface and manage user-maintained external links.
- Added curated-frame export format settings with persisted format selection, unified export entry points, and JPG export support.
- Refined the player experience with a more edge-to-edge layout, page-level idle chrome hiding, and lightweight feedback for seek / curated actions while chrome is hidden.
- Added library path reveal support and related backend playback / scan wiring improvements.
- Synced API / README / CLAUDE / project facts documentation with the new runtime behavior.
