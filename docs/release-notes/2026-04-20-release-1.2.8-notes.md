# Curated v1.2.8 Release Notes Draft

## Summary

Curated v1.2.8 introduces packaged-app update checks backed by GitHub Releases, making it easier to see when a newer Windows build is available and jump directly to the official release page.

## Highlights

- Added Settings -> About update checking against the latest GitHub Release.
- Added a lightweight sidebar update badge / compact dot when a newer installer is available.
- Added backend update-status persistence in SQLite so update check results can be reused across the app.
- Added installer/runtime version fields to backend health and app-update API contracts.
- Added localized update-check UI strings for English, Simplified Chinese, and Japanese.
- Published Windows release artifacts through the production packaging flow.

## Suggested GitHub Release Body

```md
## Curated v1.2.8

This release adds built-in update checking for packaged Windows builds. Curated can now compare the installed app version with the latest GitHub Release, show update status in Settings -> About, and surface a small sidebar badge when a newer release is available.

### Highlights

- Added update checking in Settings -> About, including current package version, latest GitHub Release version, status details, and a direct link to the official release page.
- Added a lightweight sidebar update indicator: full sidebar shows a `New` badge, compact sidebar shows a small dot, and clicking the brand area routes to Settings -> About.
- Added backend support for app-update status with GitHub Release metadata fetching, SQLite-backed status caching, and API endpoints for cached status / forced refresh.
- Extended backend health and version data with installer/runtime package metadata.
- Added localized UI copy for English, Simplified Chinese, and Japanese.

### Downloads

- Portable: `Curated-1.2.8-windows-x64.zip`
- Installer: `Curated-Setup-1.2.8.exe`

### SHA256

- `Curated-1.2.8-windows-x64.zip`: `08CC1A94000714B63A8F6C19FE5044A97D59669D36534B6A455F4AC295D4E079`

### Notes

- Update checking only reports availability and opens the official GitHub Release page. Curated does not auto-download or silently install updates.
- If you use the portable package, extract it to a clean folder before running `curated.exe`.
```
