# Curated v1.4.2

This release focuses on update delivery, controller support, player feedback polish, homepage refresh stability, and documentation for the next desktop packaging direction.

Build stamp: `20260502.165208`

## Highlights

- Added Settings -> About support for downloading the latest `.exe` installer directly from GitHub Releases when a release exposes an installer asset.
- Added a player settings switch for gamepad controls, shared gamepad input utilities, global focus navigation, library-grid navigation, and player gamepad controls.
- Added larger controller seek jumps and kept player action feedback visible when the player toolbar is hidden.
- Added right-top hidden-toolbar feedback for pause/resume and volume up/down actions, matching seek feedback behavior.
- Improved homepage recommendation refresh behavior so refreshes preserve the hero, keep recommendation IDs stable, and visibly slow the refresh spinner.
- Hardened notification-center behavior and documented the follow-up optimization plan.
- Added Electron desktop migration planning notes, including the lighter browser-shell approach for future desktop packaging work.
- Serialized health-check tests that mutate installer-version globals to avoid release-gate flakiness.
