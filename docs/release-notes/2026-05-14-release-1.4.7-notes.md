# Curated v1.4.7

Curated v1.4.7 makes the Electron desktop shell the default installed Windows app, packages the Go backend under Electron resources, and fixes a library-watch path collision that could hide newly importable files until they were moved to another directory.

Build stamp: `20260513.192311`

## Highlights

- The Windows installer and portable package now launch `Curated.exe` as the Electron desktop shell.
- The release Go backend is bundled at `resources/app/curated.exe` and is started by Electron with `-mode http` when Electron owns the backend lifecycle.
- Production frontend files are bundled at `resources/app/frontend-dist`, and Electron loads the backend-hosted Web UI on `http://127.0.0.1:8081`.
- FFmpeg runtime files are bundled under `resources/app/third_party/ffmpeg/bin`, matching the backend working directory used by the packaged Electron app.
- The installer shortcuts, post-install launch action, main window, and tray now use the Curated desktop shell path and packaged icon path.
- Fixed library watch scan collisions so newly added importable files are not incorrectly skipped when configured library paths share overlapping or colliding path state.
