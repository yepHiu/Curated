# Curated v1.2.7 Release Notes Draft

## Summary

Curated v1.2.7 focuses on playback reliability and homepage freshness, while also moving the Windows packaging flow onto the new Python-based release pipeline.

## Highlights

- Fixed player seek and timeline synchronization issues in the in-page player.
- Improved keyboard seek behavior after using the mouse on the playback progress bar.
- Refined Direct vs HLS seek waiting behavior so Direct playback no longer shows HLS-style seek-buffering messaging.
- Improved homepage daily recommendations freshness with rolling recent-history exclusions and graceful fallback windows.
- Switched the production packaging workflow to the Python release CLI and refreshed package build history / manifest generation.

## Curated v1.2.7

This release focuses on playback reliability, homepage recommendation freshness, and the new Python-based Windows packaging pipeline.

- Fixed in-page player seek and timeline synchronization issues.

  - the backend now prefers to avoid titles surfaced in the last 7 days,
  - and degrades gracefully through `7 -> 5 -> 3 -> 1 -> 0` day exclusion windows when inventory is limited.
- Migrated the Windows release packaging flow to the Python release CLI and refreshed release manifest generation.
