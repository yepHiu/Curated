# Curated v1.3.2 Release Notes Draft

## Summary

Curated v1.3.2 focuses on player reliability and playback controls. This release adds an in-player playback settings menu for speed and Direct/HLS switching, improves HLS startup buffering after mode switches, bundles FFmpeg with the Windows package for HLS playback, enlarges slider hit areas, and tightens sidebar navigation structure.

## Highlights

- Added a playback settings menu in the player chrome with speed selection and Direct/HLS playback mode switching.
- Improved Direct -> HLS switching by resetting the video element pipeline before HLS takeover and starting hls.js from the generated session origin.
- Smoothed HLS startup by shortening generated HLS segments to two seconds and giving the second segment a bounded readiness window before exposing the session.
- Bundled real FFmpeg and FFprobe binaries into Windows release packages under `third_party/ffmpeg/bin/`, avoiding production fallback to Scoop shims or missing PATH dependencies.
- Improved playback settings menu readability on bright video frames with a high-contrast dark floating surface while preserving the pink primary selected state.
- Enlarged the player slider interaction area while keeping the visual track slim.
- Refined sidebar navigation section separation and aligned the settings entry with the same link structure as the main navigation items.

## Curated v1.3.2
