# Curated v1.4.4

Curated v1.4.4 focuses on settings-page polish, safer comment editing, fuller in-app release notes, storage-health visibility, and the first MIT-licensed open-source package metadata.

Build stamp: `20260511.170115`

## Highlights

- Added automatic saving for the detail-page “My comment” editor, including debounce, in-flight save queuing, unsaved/saved feedback, and tests for long comments and readonly mode.
- Updated the in-app update panel so expanded release notes can show the full GitHub Release body, refresh stale title-only cache entries, and preserve release-note text when silent refreshes fail.
- Refined settings-page card titles, hints, nested rows, action buttons, badges, select alignment, and playback seek-step controls for denser and more consistent scanning.
- Added About-page authorship, repository, and MIT license information, plus the root `LICENSE` file and `package.json` MIT license metadata.
- Standardized production release-note rules so GitHub Release copy is publish-ready body text with final wording.
- Included the storage status alert and notification work shipped since the previous package in this production build.
