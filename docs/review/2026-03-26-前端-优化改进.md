# 前端优化改进文档 (2026-03-26)

## 一、UI/UX 设计问题与改进建议

### 1.1 交互反馈优化

#### 问题：操作反馈不一致
**位置**: `DetailView.vue`, `LibraryView.vue`

**现状**:
- 收藏、评分等操作只有错误提示，没有成功反馈
- 删除操作有 loading 状态，但其他操作没有
- Toast 提示仅在扫描任务完成时出现

**改进建议**:
```typescript
// 添加操作成功反馈
const toggleFavorite = async (payload) => {
  try {
    await libraryService.toggleFavorite(payload.movieId, payload.nextValue)
    pushAppToast(t('detail.favoriteSuccess'), { variant: 'success' }) // 添加成功提示
  } catch (err) {
    pushAppToast(formatClientError(err, t('detail.errFavorite')), { variant: 'destructive' })
  }
}
```

**优先级**: 🔴 高

---

#### 问题：缺少加载骨架屏
**位置**: `LibraryPage.vue`, `DetailView.vue`

**现状**:
- 影片列表加载时显示空白或简单文字 "Loading..."

```vue
<template v-if="loading">

```

**优先级**: 🟡 中

#### 问题：虚拟滚动列表无滚动位置记忆
**位置**: `VirtualMovieMasonry.vue`

**现状**:
- 从详情页返回列表时，滚动位置丢失
- 用户需要重新滚动找到之前的位置

**改进建议**:
```typescript
// 使用 scroll position 记忆
const useScrollPosition = (key: string) => {
  const savePosition = (pos: number) => {
    sessionStorage.setItem(key, String(pos))
  }

  const restorePosition = () => {
    const pos = sessionStorage.getItem(key)
    if (pos) scrollContainer.scrollTop = Number(pos)
  }

  return { savePosition, restorePosition }
}
```

### 1.4 搜索体验优化

**位置**: `AppShell.vue:349-392`

**改进建议**:
```typescript
// 增强键盘导航
onLibrarySearchKeydown(e: KeyboardEvent) {
  switch(e.key) {
    case 'Home':
      e.preventDefault()
      highlightIndex.value = 0
      break
    case 'End':
      e.preventDefault()
      highlightIndex.value = items.value.length - 1
      break
    case 'PageDown':
      e.preventDefault()
      highlightIndex.value = Math.min(highlightIndex.value + 5, items.value.length - 1)
      break
  }
}
```

### 2.2 焦点管理

#### 问题：Modal/Dialog 焦点陷阱缺失
**位置**: `Dialog` 组件

**现状**:
- 打开弹窗后，焦点可能跑到弹窗外
- 关闭弹窗后，焦点没有回到触发元素

**改进建议**:
```typescript
// 使用 useFocusTrap
import { useFocusTrap } from '@vueuse/integrations/useFocusTrap'

const { activate, deactivate } = useFocusTrap(dialogRef)

onMounted(() => {
  activate()
  // 保存之前焦点
  previousFocus.value = document.activeElement
})

onUnmounted(() => {
  deactivate()
  // 恢复焦点
  previousFocus.value?.focus()
})
```

**优先级**: 🟡 中
